# 🌍 PixelWorld - Monde Infini Multijoueur

Jeu multijoueur avec carte infinie où tu peux construire, dessiner et créer des alliances !

## 🚀 Installation Ultra Rapide

```bash
npm install
npm start
```

Ouvre : **http://localhost:3000**

## 🎮 Caractéristiques

- 🗺️ **Carte infinie** - Explore sans limites
- 🎨 **Dessine** sur tes territoires
- 🤝 **Alliances** - Travaille avec d'autres joueurs
- 💾 **Sauvegarde auto** - Ton progrès est gardé (cookies)
- 📍 **Coordonnées** - Retrouve tes amis facilement
- ⚔️ **Construis & Détruis** - Deux rôles en un

## 🎯 Comment Jouer

1. **Explore** - Clique pour te déplacer
2. **Construis** - Place des blocs sur la carte
3. **Dessine** - Colorie tes territoires
4. **Allie-toi** - Crée des alliances avec d'autres joueurs
5. **Coordonnées** - Utilise les coords en haut pour naviguer

## 🛠️ Technologies

- **Backend** : Node.js + Express + Socket.IO + SQLite
- **Frontend** : HTML + CSS + Canvas
